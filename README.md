📊 Financial Stakeholder Insights & Performance Dashboard
🚀 Overview

The Financial Stakeholder Insights and Performance Dashboard is an interactive Tableau project designed to analyze financial data and uncover meaningful insights related to stakeholders, insurers, and policy performance.

This project focuses on transforming raw data into clear, actionable insights through intuitive and visually engaging dashboards.

🎯 Objective
Analyze premium trends and growth patterns
Evaluate insurer performance and value creation
Identify profitability drivers across regions and policies
Enable data-driven decision-making through visualization
🛠️ Tools & Technologies
📊 Tableau (Data Visualization)
🐍 Python (Data Understanding)
🗄️ Excel / CSV (Dataset)
📁 Dataset

The dataset includes structured financial and stakeholder data such as premiums, insurers, cost ratios, demographics, and asset details.

🔗 Dataset Link:
https://github.com/Aman6393k/Financial-Stakeholder-Insights-Dashboard/blob/main/Finance%20Record%20Data%20set.xlsx

📊 Dashboard Features
🔹 Key Visualizations
Premium trends over time
Insurer-wise performance comparison
Shareholding distribution
Cost ratio vs profitability
City-wise and tenure-based insights
Asset distribution across stakeholders
📈 Key Performance Indicators (KPIs)
Total Premium Collected
Average Stakeholder Age
Total Value Created
Top Performing Insurer
🎛️ Interactivity
Dynamic filters (City, Gender, Stakeholder Type)
Policy tenure selection
Date range filtering
Interactive charts for deeper exploration
