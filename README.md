# 📊 Financial Stakeholder Insights & Performance Dashboard

## 🚀 Overview

The **Financial Stakeholder Insights and Performance Dashboard** is an interactive data visualization project built using Tableau to analyze and uncover key business insights related to stakeholders, insurers, and policy performance.

This project transforms raw financial and policy data into meaningful, actionable insights, enabling stakeholders to make informed, data-driven decisions.

---

## 🎯 Objective

The goal of this project is to:

* Analyze premium trends and business growth patterns
* Evaluate insurer performance and value creation
* Understand profitability drivers across regions and policies
* Provide a centralized view of stakeholder related insights

---

## 🛠️ Tools & Technologies

* 📊 Tableau (Data Visualization)
* 📁 CSV / Excel (Dataset)

---

## 📁 Dataset

The dataset contains structured financial and stakeholder-related data, including premiums, insurers, cost ratios, demographics, and asset details.

🔗 **Dataset Link:** *(https://github.com/Aman6393k/Financial-Stakeholder-Insights-Dashboard/blob/main/Finance%20Record%20Data%20set.xlsx)*

---

## 📊 Dashboard Features

### 🔹 Key Visualizations

* Premium trends over time
* Insurer-wise value creation comparison
* Shareholding distribution
* Cost ratio vs profitability analysis
* City-wise and tenure-based profitability insights
* Asset distribution across stakeholders

---

## 📈 Key Performance Indicators (KPIs)

* Total Premium Collected
* Average Stakeholder Age
* Total Value Created (Equity-based assets)
* Highest Value Created by Insurer

---

## 🎛️ Filters & Interactivity

* City-wise filtering
* Stakeholder-based analysis
* Policy tenure range slider
* Date range selection
* Gender-based insights

✨ Interactive dashboard where clicking on visuals dynamically updates all charts and KPIs.
